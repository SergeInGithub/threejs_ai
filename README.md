# 🎭 Three.js_Ai

## ℹ️ Description
This A full stack web app that generates personalized t-shirts with Ai, solid colors or image uploads as a t-shirt Logo or full texture

## 🧬 Live Site

https://threejs-ai-nine.vercel.app/

## 🔮 Tech Stack

- Artificial intelligence
- Three.js
- React
- Typescript
- Node.js
- Express
- Mongoose

## 📦 Installation

```bash
pnpm install
```

## 🚀 Running the app

```bash
# development
pnpm dev
```
